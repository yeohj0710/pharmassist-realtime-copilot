# Legal, privacy, and clinical review checklist

Release 승인은 코드 통과만으로 끝나지 않는다.

- [ ] 공식 의약품/DUR/허가 source별 사용권·재배포 조건 기록
- [ ] 모든 published card의 A/B source·locator·verified/expiry 확인
- [ ] 약사와 medical-safety reviewer 이중 승인 기록
- [ ] reviewer/publisher 분리 및 실제 OIDC role 검증
- [ ] 개인정보 처리방침, PIA, 동의·고지·보유기간 승인
- [ ] 국외 이전과 OpenAI retention/ZDR/MAM/계약 조건 승인
- [ ] 한국 의료·약사·의료기기·광고 관련 규제 법률 검토
- [ ] target 약국 기기/소음/방언/고령층 접근성·음성 평가
- [ ] 장애·revocation·rollback 교육과 drill 완료
- [ ] synthetic/placeholder 0건과 signing key custody 확인

서명자, 날짜, evidence URL, 예외와 만료일을 release record에 남긴다.
