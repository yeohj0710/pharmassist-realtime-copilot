## Release boundary

애플리케이션 구현과 합성 데모 검증은 자동화한다. 공식 의약품/DUR 데이터 사용권, 약사 이중 검수, 한국 개인정보·규제 검토, OpenAI 조직 데이터 통제, 실제 약국 기기·음성 평가는 외부/manual gate로 남는다. 이 gate 전에는 임상 production-ready로 표시하지 않는다.
