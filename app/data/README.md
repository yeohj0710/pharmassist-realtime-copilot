# Data boundary

`synthetic/` and `generated-dev-pack/` are DEMO ONLY. They are not official clinical data and are prohibited from clinical production use. Raw patient audio, transcripts, source dumps, and signing private keys must never enter this repository.

Production activation requires an approved official-source adapter, recorded license/terms, immutable locator, A/B trust tier, current verification/expiry, pharmacist and medical-safety approval, publisher separation, and an external private signing key.
