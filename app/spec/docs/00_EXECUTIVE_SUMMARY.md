# 00. Executive Summary

## 제품

약사가 환자의 발화나 짧은 키워드를 입력하면 **즉시 말할 문장, 다음 질문, 위험신호, 검증된 주의사항**을 보여주는 카운터용 실시간 코파일럿이다.

## 기술 방향

- 자체 foundation model 개발 또는 로우레벨 학습이 아니다.
- 승인된 지식을 미리 상담카드로 컴파일한다.
- 브라우저 로컬 검색을 실시간 주 경로로 사용한다.
- OpenAI Realtime API는 부분 음성 전사에 사용한다.
- OpenAI Responses API는 선택적 문장 보정·모호성 해소에 사용한다.
- 의료 사실과 안전 분기는 LLM이 아니라 버전 관리되는 claim/rule/card에 둔다.

## 1초에 대한 정의

1초 내에 **완성된 모델 답변**이 아니라 **첫 유용 상담카드**를 표시한다. 타이핑 exact/rule은 네트워크 없이 P95 250ms 이하를 목표로 한다. 음성은 안정된 부분 전사가 도착한 뒤 P95 700ms 이하를 목표로 한다. 임의 네트워크에서 음성 end-to-end 1초는 보장하지 않는다.

## 자료 사용 결론

첨부 자료에는 실무 상담 흐름과 표현이 풍부하지만, 판촉·비표준 체질론·근거가 불명확한 영양소 주장·최신 제품정보가 필요한 고위험 규칙이 혼재한다. 따라서 자료 전체를 런타임 RAG에 넣지 않고, 후보 카드 추출→공식 출처 검증→약사 승인 과정을 통과한 claim만 배포한다.

## 최초 배포 범위

- 한국어, 사람 OTC, 약사 전용
- 타이핑 우선, push-to-talk 음성은 다음 기능 플래그
- 진단·처방·환자 직접대화 없음
- 원본 음성/전사 저장 없음
- 소아 용량·임부·상호작용 등은 공식 데이터가 연결되고 검수된 카드만
- 동물약·건기식·약국 운영은 별도 도메인으로 기본 비활성

## 성공 지표

- 첫 카드 지연시간
- 위험신호 재현율
- 필수정보 누락 시 안전 차단률
- 약사 수용/수정/거절률
- ASR 약학 용어 오류율
- 출처 만료·충돌·미승인 claim 0건
- 개인정보 원문 로그 0건
