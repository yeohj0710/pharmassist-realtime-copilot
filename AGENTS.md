# PharmAssist 상담 UX 원칙

- 환자 대사는 실제 약국 약사가 짧게 말하는 자연스러운 한국어로 작성한다.
- 연령, 임신·수유, 알레르기, 주요 질환, 복용약, 증상 기간을 한 문장에 전부 묻는 보험성 질문을 환자 화면에 노출하지 않는다.
- 내부 작성 지침, 엔진 용어, 스키마 설명, 안전 체크리스트를 환자 대사로 출력하지 않는다.
- 추가 확인이 필요하면 현재 증상과 추천 선택을 실제로 가르는 질문 한 가지만 묻는다.
- 안전 규칙과 red flag 판정은 유지하되, 약사용 내부 점검과 환자에게 할 말을 분리한다.
- 위험 신호가 명확한 경우에만 진료·응급 안내를 표시한다.
- 질문 중에도 임상적으로 구분 가능한 현재 후보가 있으면 적용 조건과 함께 먼저 보여준다.
